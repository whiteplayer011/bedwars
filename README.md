# MeuPlugin

Este é um plugin de BedWars para Minecraft 1.8.8, inspirado no servidor MushMc.

## Requisitos
- Spigot 1.8.8
- Java 8

## Instalação
1. Compile o projeto utilizando Gradle.
2. Coloque o arquivo JAR gerado na pasta `plugins` do seu servidor Spigot 1.8.8.
3. Inicie o servidor.

## Comandos
- `/bedwars`: Comando principal do plugin.
